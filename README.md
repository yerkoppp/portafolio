## ♿ Accesibilidad Neurodivergente

Este portfolio implementa un modo de accesibilidad diseñado siguiendo las pautas WCAG AA/AAA:

| Característica | Descripción |
|---|---|
| **Tipografía** | Fuente OpenDyslexic con mayor espaciado entre letras y palabras |
| **Movimiento** | Todas las animaciones y transiciones desactivadas |
| **Contraste** | Paleta de colores de alto contraste verificada |
| **Alineación** | Textos alineados a la izquierda para facilitar la lectura |
| **Persistencia** | Preferencia guardada en `localStorage` entre páginas |

Activable desde **Ajustes** en el simulador Android o desde el navbar en la versión Web.

## 🔒 Easter Egg — Modo Desarrollador

El simulador Android incluye un easter egg inspirado en el modo desarrollador nativo de Android:

1. Abrir **Ajustes**
2. Tocar **"Versión de Android"** 5 veces seguidas
3. Se desbloquea la **Terminal** oculta con efecto Matrix

La Terminal acepta comandos reales: `help`, `about`, `skills`, `contact`, `clear`, `exit`.
